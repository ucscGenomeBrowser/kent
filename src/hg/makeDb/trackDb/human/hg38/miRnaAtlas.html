Description

The Human miRNA tissue atlas is a catalogue of miRNA tissue specific expression across 62 various tissues. This track contains quantile normalized miRNA expression data sampled from two individuals. The track contains two subtracks, one for each individual sampled.

The Tissue Specificity Index (TSI) is analagous to the "tau" value for mRNA expression, and is calculated as described in the associated publication. Values closer to 0 indicate miRNAs expressed in many or all tissues, while values closer to 1 indiciate miRNAs expressed only in a specific tissue or tissues. To browse miRNAs by TSI value, please see the miRNA tissue atlas.

Data Access

The underlying expression matrix and TSI values can be obtained from the miRNA tissue atlas website, in the data_matrix_quantile.txt and tsi_quantile.csv files.
