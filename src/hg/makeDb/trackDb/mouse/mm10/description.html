Assembly details

Genome Reference Consortium GRCm38, which includes approximately 2.6 Gb of sequence, is considered to be "essentially complete". The assembly includes chromosomes 1-19, X, Y, M (mitochondrial DNA) and chr*_random (unlocalized) and chrUn_* (unplaced clone contigs). For information about the process used to assemble this version, see the GRC website. For a glossary of assembly-related terms, see the GRC Assembly Terminology page. Note that the UCSC mm10 database contains only the reference strain C57BL/6J.
