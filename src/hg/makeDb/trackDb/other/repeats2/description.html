RepeatMasker Consensus Sequences
 (GIRINST)

This is an experimental browser for consensus sequences built from the RepeatMasker track in hg19.

Search the assembly:

  • By position or search term: Use the "position or search term" box to find areas of the genome associated with many different attributes, such as a specific chromosomal coordinate range; More information, including sample queries.
  • By track type: Click the "track search" button to find Genome Browser tracks that match specific selection criteria. More information.
