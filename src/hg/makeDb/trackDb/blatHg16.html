Description

This track displays alignments of the $o_date $o_Organism genome vs. the $Organism genome.

Methods

The alignments were done with BLAT in translated protein mode using the parameters -q=dnax -t=dnax. The default settings were used otherwise. Both genomes were masked with RepeatMasker and Tandem Repeat Finder before running BLAT.
