Sample position queries

A genome position can be specified by a scaffold coordinate range, the accession number of an mRNA or RefSeq entry, a gene name, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the A. mellifera genome. Note that some position queries (e.g. "huntington") may return matches to the mRNA records of other species. In these cases, the mRNAs are mapped to their homologs in A. mellifera. See the User's Guide for more information.

Assembly details

Amel_1.2 is a whole genome shotgun assembly produced using Baylor HGSC's assembly engine, Atlas. The assembly, which provides approximately 6x coverage of the clonable portion of the genome, contains 9751 scaffolds. Scaffolds are named according to 17 linkage groups (Group1.* - Group16.* and GroupUn.*). The total scaffold size for this assembly is 213 Mb including gaps between contigs. Total contig size is 208 Mb. The median scaffold size is 2,189 bp, average size is 21,884 bp, and the largest scaffold is 2,167,338 bp. The N50 scaffold size, defined as the length such that 50% of the assembled genome lies in blocks of the N50 size or longer, is 260 kb. More details about Amel_1.2 can be found here.

Bulk downloads of the sequence and annotation data are available via the Genome Browser FTP server or the Downloads page. These data are made available with specific conditions for use. The apiMel1 annotation tracks were generated by UCSC and collaborators worldwide.
