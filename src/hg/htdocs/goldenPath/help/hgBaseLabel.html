Labeling EST bases
 

The Genome Browser's base-labeling feature allows users to display all bases in the aligning sequence or only those that differ from the genomic sequence. This permits quick validation and comparison of EST sequences.

To turn on base-labeling, select the desired option from the Show sequence for ... bases pull-down menu:

  • all bases - Labels all bases in the EST sequence.
  • different bases - Labels only the EST bases that differ from their counterparts in the genome sequence.
  • no bases - Hides the base labels.

When an alignment error condition is encountered, the alignment block is colored yellow. If you find one of these, please contact the UCSC Genome Browser team to report it.

When are the EST bases labeled?
To view the labeled bases, the option must be turned on, the track must be in squish, pack, or full display mode, and the browser must be zoomed-in to a level at which there are enough pixels per base for the text size in use. The zoom-level requirement varies with the image width of the browser.
