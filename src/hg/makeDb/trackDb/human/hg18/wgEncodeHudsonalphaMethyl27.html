HudsonAlpha Methylation27

Description

These tracks display the methylation status of specific CpG dinucleotides in the given cell types as identified by Illumina Infinium HumanMethylation27 BeadArray platform. The Methyl27K platform uses bisulfite treated genomic DNA to assay the methylation status of specific CpG dinucleotides. Genomic DNA treated with sodium bisulfite converts unmethylated cytosine of a CpG dinucleotides into uracil; methylated cytosines do not get converted. After bisulfite treatment, the methylation status of a site is assayed by single base-pair extension with a Cy3 or Cy5 labeled nucleotide on oligo-beads specific for the methylated or unmethylated state. A beta value is calculated by Illumina's Bead Studio software for each CpG target. This value represents the intensity value from the methylated bead type divided by the sum of the intensity values from the methylated and unmethylated bead types for any given CpG target.

This method was used to validate Methyl-seq data generated in collaboration with the laboratory of Dr. Julie Baker at Stanford University to study methylation and gene expression changes that occur in human embryonic stem cells before and after differentiation to definitive endoderm. Based on the results of these experiments, cut-off beta values to call "methylated" and "unmethylated" are selected. Any beta value equal to or greater than 0.6 is considered fully methylated. Any beta value equal to or less than 0.2 is considered to be fully unmethylated. Beta values between 0.2 and 0.6 are considered to be partially methylated. Beta-values are quality filtered and spots that fall below the minimum intensity threshold are displayed as "NA".

Scores associated with each site are beta value multiplies by 1000. Methylation status are color-coded as:

  • red = methylated
  • purple = partial methylated
  • bright blue = unmethylated
  • black = NA

The data sets we submit are in a tab-delimited format with Illumina CpG Identifier, chromosome number, genomic coordinate, gene symbol for the nearest gene, and beta value for each cell line, and we perform the experiment for two replicates for each cell line. Detailed information for the CpG targets are on Illumina's website: http://www.illumina.com/downloads/Illumina_HumanMethylation27_v1_Content.zip

Methods

Isolation of genomic DNA

Genomic DNA is isolated from biological replicates of each cell line by using the QIAGEN DNeasy Blood & Tissue Kit according to the instructions provided by the manufacturer. DNA concentrations and a level of quality of each preparation is determined by UV absorbance.

Sodium Bisulfite Conversion

Bisulfite conversion reaction was done using the Zymo Research EZ-96 DNA MethylationTM Kit protocol. One step of the protocol was modified. During the incubation, we included a 30 sec 95oC denaturing step every hour to increase reaction efficiency as recommended by the Illumina Infinium Human Methylation27 protocol.

Infinium HumanMethylation27 BeadArray

The bead arrays were ran according to the protocol provided by Illumina.

Processing and analysis software used

The intensity data from the BeadArray was processed using Illumina's BeadStudio software with the Methylation Module v3.2. The data was then quality-filtered using p-values.
