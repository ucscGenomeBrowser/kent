Description

This track shows DNase I Footprint data from FlyReg, version 1.0. FlyReg provides access to results of the systematic curation and genome annotation of 1,350 DNase I footprints for the fruitfly D. melanogaster reported in Bergman, Carlson and Celniker (2004) Bioinformatics (in press).

When available, a footprint motif is also displayed, based on a MEME matrix computed by Dan Pollard on the set of footprints for this factor.
