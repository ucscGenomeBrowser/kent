ENCODE Registered Experimental Variables

To facilitate integration of data between the contributing research groups, the ENCODE Consortium has implemented a controlled vocabulary. This page lists the registered experimental variables that make up the ENCODE controlled vocabulary. Before submitting data to the DCC, all controlled vocabulary terms used in ENCODE data submissions must be registered.


These three registered experimental variables have their own pages, which also have direct links from the ENCODE home page:

  • Antibodies
  • Human Cell Types
  • Mouse Cell Types
