Description

This track shows expression data from the Zon Lab at Children's Hospital Boston using the Affymetrix Zebrafish GeneChip Genome Array. The chip type or strain medians associated with replicate experiments (Chip ID display) or the average of this data across all strains for each developmental stage (Tissue Averages display) can be displayed by selecting the appropriate option from the Experiment Display menu on the track description page. For more information, see the Track Configuration section.

Methods

For detailed information about the experimental methods, see Weber et al. (2005) below. This expression data is for five strains of wild type zebrafish whole embryos at different developmental stages. Alignments displayed on the track correspond to the target sequences used by Affymetrix to choose probes.

Microarray data were preprocessed using MAS background correction, MAS probe specific correction, and Loess normalization. Bioconductor's affy package was used to perform this processing. CEL files were normalized as a single group and log2 transformed with scaling so that the average (mean) value of the expression data is zero for each single experiment. The track color denotes the average (median) signal over all replicate experiments for a specific developmental stage and strain on a log base 2 scale. Lighter colors correspond to lower signals; darker colors correspond to higher signals. Each row on the track or details page shows gene expression data for a single developmental stage and a single strain (see Strains and Stages table below) from wild type, whole embryo zebrafish. Data was collected by the Children’s Hospital Boston Zebrafish Genome Initiative.

The rows show the median expression of a gene for a specific developmental stage and strain. Over-expressed genes are colored red; under-expressed genes are show in green. The values are colored on a logarithmic scale. Black indicates that a gene is neither over- nor under-expressed in the tissue. Uncolored boxes (white on most browsers) represent missing data.

Track Configuration

This track may be configured to change the display mode and colors or vary the type of experiment information shown. The configuration controls are located at the top of the track description page, which is accessed via the small button to the left of the track's graphical display or the link on the track's control menu.

  • Display mode: To change the display mode for the track, select the desired display setting from the Display Mode pulldown list.
  • Experiment Display: To display the chip type, tissue averages, or individual chip identification numbers associated with the track items in the current position range, select the corresponding item from the Experiment Display pulldown list.
  • Color scheme: To change the colors with which the track is displayed, click the button in front of the desired color scheme.
