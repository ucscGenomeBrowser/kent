Search Tools for ENCODE Data at UCSC

This page contains links to search tools for finding tracks displayed in the browser or downloadable files from the ENCODE project at UCSC during the first ENCODE production phase (Sep 2007 - Jun 2012). For bulk download, retrieval by FTP is recommended along with rsync. Using rsync, for example "rsync -a -P rsync://hgdownload.gi.ucsc.edu/goldenPath/hg19/encodeDCC/wgEncodeDir/wgEncodeFile ./", has the advantage of starting up where it left off after a failure, when ran again.
