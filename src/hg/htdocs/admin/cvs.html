To update an existing source tree, use the command: cvs update -A -d -P -rbeta. You will want to update your source tree at least once every two weeks. The Genome Browser is under continual development and often has significant enhancements and new features added on a regular basis. A new beta tag marks a new release of the code usually every two weeks.

After an update the source tree can be rebuilt with the sequence:

  • cd kent/src
  • make clean
  • make cgi
The 'make clean' step is important.

See also: the README files in the source tree directory src/product/README.*
