  Repeatmasking of danRer2 chr1
 

An error was found with the RepeatMasker output for chr1 on the Zebrafish (danRer2) assembly. This resulted in an incomplete masking of the repeats in the chr1.fa. This error has been corrected for the data in the chr1.fa downloads file. However, blastz has not been rerun on chr1, which may result in some inconsistencies.
