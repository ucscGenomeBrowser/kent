Description

This track shows preliminary gene homology-based synteny blocks for the $organism assembly and the rat assembly (June 2003 - rn3). See http://azra.embl-heidelberg.de/~zdobnov/Chicken/ for a summary and description as well as links to the maps.

Methods

See http://azra.embl-heidelberg.de/~zdobnov/Chicken/methods.txt.
