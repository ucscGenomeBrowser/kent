Contact information for the UCSC Genome Project
 

Genome Browser contacts:
We maintain two public interactive mailing lists (genome@soe.ucsc.edu and genome-mirror@soe.ucsc.edu) where you can post specific questions about the UCSC Genome Browser software, database, genome assemblies, release cycles, and mirroring. For more information on the genome-mirror mailing list, see the Mirroring section below. We also offer a third read-only mailing list reserved for project announcements posted by our staff.

Prior to posting a question to the mailing list, please review our technical support guidelines. Many questions have been answered before; we strongly encourage you to search the archives before submitting a question to either list.

  • Browse the Genome mailing list archives (indexed by thread, subject, author and date).
  • Browse the Genome-announce mailing list archives (indexed by thread, subject, author and date).
  • Search the Genome Browser website:  
  • Search the Genome mailing list archives:  
  • Search the Genome-announce mailing list archives:  
  • Post a question to the genome@soe.ucsc.edu mailing list. Messages sent to this address will be posted to the moderated genome mailing list, which is also archived on a public Web-accessible pipermail archive. This archive may be indexed by non-UCSC sites such as Google. When including a link to a specific view of the browser in your question, please remove the "hgsid" paramter and setting from the URL as this can cause your personal settings to change unexpectedly. We recommend taking a snapshot of your current browser view with our Session tool and sending a link to that session.
  • Subscribe to the Genome-announce mailing list (UCSC Genome Browser project announcements only - low-volume list).
  • Subscribe to the Genome mailing list (interactive Genome Browser discussion forum - high-volume list).
Reporting server problems:
Server errors or other problems accessing the UCSC Genome Browser or BLAT server should be sent to genome-www@soe.ucsc.edu.

Regular weekly maintenance of the Genome Browser database occurs on Sunday afternoons between the hours of 2:00 - 3:00 p.m. Pacific Time. Although we allocate an hour for this maintenance, the actual down time is usually only a few minutes. Occasionally, additional maintenance is scheduled by the School of Engineering. These service interruptions are posted on the Genome Browser home page.

Commerical licensing:
Questions about commercial licensing of the Genome Browser should be sent to genome-www@soe.ucsc.edu. For more information about licensing, see the Genome Browser software licensing requirements.

Questions regarding commercial licensing of Blat should be directed to kent@soe.ucsc.edu. See the Kent Informatics web site for more information.

Mirroring:
The genome-mirror mailing list is a forum to which licensed and non-commercial sites can post questions about the setup and maintenance of Genome Browser mirrors. Note that commercial use of the source code requires a license.
  • Post a mirror site inquiry: genome-mirror@soe.ucsc.edu. Messages sent to this address will be posted to the moderated genome-mirror mailing list, which is archived on a public Web-accessible pipermail archive. This archive may be indexed by non-UCSC sites such as Google.
  • Browse the Genome-mirror mailing list archives (indexed by thread, subject, author and date).
