Description

The Old UCSC Genes track shows genes from the previous version of the UCSC Genes build. This is similar to the current version is based on less Genbank, RefSeq, and UniProt data.

The new release has 85,810 total transcripts, compared with 77,614 in the previous version. The total number of canonical genes has increased from 27,389 to 28,661. Comparing the new gene set with the previous version:

  • 26,658 transcripts did not change between versions.
  • 1,010 transcripts were not carried forward to the new version.
  • 48,057 transcripts are "compatible" with those in the previous set, implying that the two transcripts show consistent splicing.
  • 1,889 transcripts overlap with those in the previous set but do not show consistent splicing, i.e. they contain overlapping introns with differing splice sites.
