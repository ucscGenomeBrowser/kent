Trix Indices
 

A Trix index consists of a pair of files that allow for fast lookup of free text associated with a list of identifiers. A Trix index is created from a single line oriented text file using the program ixIxx. Each line starts with an identifier, with the rest of the line being free text which is associated with the id.
