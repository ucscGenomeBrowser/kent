The University of Maryland UMD 3.1.1 (bosTau8) assembly is an update to the previous UMD 3.1 (bosTau6) assembly. UMD 3.1 contained 138 unlocalized contigs that were found to be contaminants. These have been suppressed in UMD 3.1.1. For more information, see the "comment" section at http://www.ncbi.nlm.nih.gov/nuccore/DAAA00000000.2/.
