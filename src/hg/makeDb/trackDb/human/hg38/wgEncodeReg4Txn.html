Description

This track displays genome-wide transcription levels of coding and non-coding RNAs measured by total RNA-seq from ENCODE experiments. The signal represents reads per million (RPM) of uniquely mapped reads, shown separately for the plus and minus genomic strands. Each subtrack corresponds to an organ or tissue average of individual RNA-seq experiments. These datasets provide a broad view of transcriptional activity across the genome. Additional datasets measuring transcription levels are available at the ENCODE portal.

This track displays strand-specific total RNA-seq signals averaged across biosamples from the same organ or tissue. Two versions of each subtrack are available:

  • Tissues and Primary Cells Only: Restricts to total RNA-seq data from tissues and primary cells, focusing on biosamples that more closely reflect in vivo cell states and developmental contexts.
  • All Biosamples: Includes total RNA-seq data from all biosample types -- tissues, primary cells, cell lines, in vitro differentiated cells, and organoids -- providing a comprehensive view of transcription across diverse biological systems.

In both versions, RNA-seq signal is averaged across biosamples within the same organ/tissue category and displayed using a transparent overlay. For organs or tissues with only tissue or primary cell experiments, a single subtrack is shown because the two versions would be identical. For organs or tissues with only experiments from cell lines, in vitro differentiated cells, or organoids, a single all-biosample subtrack is shown. For organs or tissues that include both types, both versions are provided.

Available Organs and Tissues

Each organ or tissue is represented by two subtracks: one for the plus genomic strand and one for the minus strand. Subtracks prefixed with "All" include data from all biosample types (tissues, primary cells, cell lines, in vitro differentiated cells, and organoids), while subtracks without the prefix are restricted to tissues and primary cells only.

Display Conventions and Configuration

This track uses a transparent overlay to visualize data from multiple organs or tissues within the same vertical space. For each organ or tissue, signals from all associated experiments were averaged to generate the displayed track. Each organ or tissue is assigned a distinct color, selected to be light and saturated to maintain clarity when overlaid. Initially, each layered track displays an overlay of five representative organs: blood, brain, kidney, liver, and muscle. Clicking on the track opens a details page where you can view and select organs or tissues. Subtracks can be further filtered by strandedness (+ strand or - strand) and life stage of the biosample.

Data Access

The data for this track can be accessed from the Table Browser or combined with other datasets through the Data Integrator. The original data files are also available from the ENCODE portal.
