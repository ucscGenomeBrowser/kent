Mouse Downloads

Overview

This page contains links to directories containing raw and processed data for mouse ENCODE data produced as part of the ENCODE production phase (September 2007-present). Formats are described on the UCSC Browser File Format FAQ. For bulk download, retrieval by FTP is recommended. All files here are covered by the ENCODE data release policy.
