Sample position queries

A genome position can be specified by the accession number of a sequenced genomic region, an mRNA or EST, a chromosomal coordinate range, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the Marmoset genome. See the User's Guide for more information.

Request:   Genome Browser Response:
 
Contig12371   Displays all of contig 12371
Contig3759:130,210-140,209   Displays ten thousand bases of contig 3759
Contig2815:171328+1500 Displays a region of Contig 2815 that spans 1500 bases, starting with position 171328
 
EF215092   Displays region of mRNA with GenBank accession number EF215092
EH380513   Displays region of EST with GenBank accession EH380513
 
mRNA helicase   lists mRNA helicase
homeobox caudal   Lists mRNAs for caudal homeobox genes
zinc finger   Lists many zinc finger mRNAs
kruppel zinc finger   Lists only kruppel-like zinc fingers
misra   Lists mRNAs deposited by scientist named Misra
Taguchi,T.   Lists mRNAs deposited by co-author T. Taguchi
 
Use this last format for author queries. Although GenBank requires the search format Taguchi T, internally it uses the format Taguchi,T..

Assembly details

This is a 6X coverage assembly. There are 49,724 supercontigs with a total of 3,029,401,840 bases. The largest contig is Contig1, with a size of 2,551,648 bases. Half of the sequence is contained in the 2,807 largest contigs of size 303,219 bases (N50) or larger. The average contig size is 60,924 bases, the median contig size is 8.789 bases. There are 154,810 gaps of size 100 bases or more for a total of 98,876,036 bases. The largest gap, 3,000 bases, occurs 1,043 times. See also: WUSTL Callithrix jacchus assembly details.

Bulk downloads of the sequence and annotation data are available via the Genome Browser FTP server or the Downloads page. These data have specific conditions for use.
