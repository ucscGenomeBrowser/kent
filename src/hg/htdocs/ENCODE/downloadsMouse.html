Mouse ENCODE Downloads

This page contains links to directories containing raw and processed data for mouse ENCODE data produced as part of the ENCODE production phase (September 2007-present). For bulk download, retrieval by FTP is recommended. All files here are covered by the ENCODE data release policy.

Preview of unreleased data for download is available from the Test download server and by FTP. Preview data is made available with the warning that it has not yet been reviewed by UCSC for correctness and is subject to change at any time.
