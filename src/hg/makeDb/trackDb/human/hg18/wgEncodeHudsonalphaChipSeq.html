HudsonAlpha/Caltech ChIP-Seq Docs

Description

These tracks display binding sites of the specified transcription factors in the given cell types as identified by ChIP-Seq (1). The ChIP-Seq method uses formaldehyde to cross-link DNA associated proteins to genomic DNA within living cells. The cross-linked cells are lysed and sonicated, releasing small soluble chromatin fragments. The chromatin fragments bound by a specific or general transcription factors are immunoprecipitated by using an antibody directed against a specific transcription factor of interest. After reversing the chemical cross-links, the recovered DNA fragments are ligated to specific double-stranded DNA adapters, size-selected and sequenced. Short reads of 25-35 bp are mapped to the human reference genome, generating peaks or enriched regions of high read density relative to a total input chromatin control reads.

The tracks are grouped by transcription factor targeted antibody and by cell type. Included for each cell type is a control signal, which represents the control condition where no antibody targeting was performed. Within a grouping their will be one or more of the following:

  • Raw Signal of aligned read density
  • Called Peaks which represent the sites with the greatest evidence of transcription factor binding obtained from these experiments

Methods

Chromatin immunoprecipitation (ChIP)

Cross-linked chromatin was immunoprecipitated with antibody according to the protocol posted at http://myers.hudsonalpha.org/content/protocols.html. We tested biological replicates from each cell line.

DNA library construction and sequencing

We generated libraries from DNA fragments recovered from immunoprecipitation and total input chromatin according to the protocol posted at http://myers.hudsonalpha.org/content/protocols.html. We sequenced these libraries with an Illumina Genome Analyzer (GA1) according to the manufacturer's recommendations.


Validation of ChIP-Seq

Quantitative polymerase chain reaction (qPCR) assays can be used to validate the transcription factor binding sites found using ChIP-seq. The computational algorithm QuEST (2) identifies regions of enriched read density. QuEST reports a single genomic coordinate (peak) for each enriched region. These peaks are ranked according to the ChIP-seq enrichment ratio and qPCR assays are used to validate the set of overlapping peaks between replicates for each cell line. We designed qPCR primer pairs to interrogate the list of ordered peaks in common between replicates. Amplicons were 60-100 bp in length and were completely contained within 250 bp of either side of the peak coordinate. For each primer pair, we performed qPCR assays on biological replicate ChIP samples on both cell lines and recovered total input chromatin DNA. We calculated enrichment as a ratio of amount of target DNA over the average of a pair of negative control primers. We considered an assay as positive when it had a two-fold or greater enrichment of the average qPCR replicates (2).


Processing and analysis software used

Sequence data produced by the Illumina data pipeline software were quality filtered and then mapped to hg18 using its integrated Eland software, with multiply mapped reads discarded. To identify likely binding sites, Peak Calling was applied on the aligned sequence data sets using QuEST (2): quantitative enrichment of sequence tags (QuEST) is based on the kernel density estimation approach, which uses ChIP-Seq data to determine positions where protein complexes contact DNA. QuEST uses data in the form of genome coordinates ('tags') obtained from mapping several million sequencing reads to a reference genome. Tags from forward and reverse reads cluster on opposite sides of the transcription factor binding site. QuEST first constructs two separate profiles, one for forward and one for reverse tags. QuEST identifies candidates for combined density profile (CDP) peaks as positions in the reference genome corresponding to local maxima of the CDP with sufficient enrichment compared to the control data.
