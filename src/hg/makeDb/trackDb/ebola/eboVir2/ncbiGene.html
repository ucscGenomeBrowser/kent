Description

This track contains genes extracted from the GenBank nuccore entry for KJ660347.

Display Conventions

This track follows the display conventions for gene prediction tracks.

Methods

We downloaded the GenBank record for KJ660347.2, extracted the entries for each gene and then loaded them into the UCSC database. Additional entries were added for the various forms of the GP gene.
