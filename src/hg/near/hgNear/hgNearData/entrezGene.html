Description

This column displays the NCBI Entrez Gene ID associated with each gene, if available.

Action

Click on the ID to display the Entrez Gene record. If Entrez shows a link to the Online Mendelian Inheritance in Man (OMIM) — represented by an orange square with an "O" inside it — an OMIM record is available for the gene. OMIM is often an excellent source of human-curated information about human genes.
