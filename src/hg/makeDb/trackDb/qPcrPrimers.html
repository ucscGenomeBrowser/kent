Description

This track provides RT-qPCR primers that were automatically designed to measure the abundance of human and mouse transcripts using SYBR based qPCR. Genome assemblies mm10 version (Mouse) and hg19 version (Human) were used. The primers were generated by an automatic design procedure, to target all transcripts and all "possible" exon-exon and intron-exon junctions in the human and mouse transcriptomes.

Note that not all consecutive exon-intron-exon triplets generate "possible" primer pairs, defined as satisfying a set of imposed design rules:

  1. We do not address the first exon-exon junction, to avoid problems related to abortive transcription.
  2. Intron length should be more then 800bp to avoid problems of double products in amplification.
  3. We design only junction primers, so that forward and reverse primers must flank the junction.
  4. Melting temperature of the primers should be between 60 < Tm < 63 (optimally 60.5 according to Breslauer et al. (1986).
  5. Primer length should be 18-25bp.
  6. Product size should be 60-125bp.
  7. We first design primers for the intron-exon junction (pre-mRNA); choose the two best primer pairs, and then for the reverse primer of each pair we design two options for the corresponding forward primer of the exon-exon junction (mRNA).
  8. We use the "PRIMER MISPRIMING LIBRARY" of the primer3 software: "human" for human and "rodent" for mouse transcriptome, respectively.

The track enables the user to easily obtain primers to almost all transcripts in the transcriptome, without the need to go through the tedious process that might involve human errors. After choosing the primers the track includes a link to the primers' details, displayed in a standard format. To meet the need to design a large number of primers, we also provide a batch query website.

Methods

We use the UCSC gene model as a reference of the gene structure and the primer3 software as the design engine. The software goes over all possible exon-exon junctions in the transcriptome and applies our design rules/parameters to provide two primer pairs for every "possible" intron-exon junction and four pairs for every "possible" exon-exon junction.

Display Conventions

The resulting suggested primer pairs can be easily viewed, the primers to amplify pre-mRNA (intro-exon junctions) are shown in red and the primers to amplify mRNA (exon-exon junctions) in blue. For each pre-mRNA primer pair there are two corresponding mRNA primers (that use the same reverse primer, if possible). Each pair has a unique code which stands for the gene name and the junction name. For example, the pair "NR4A1_uc001rzt.2_1_1" amplifies pre-mRNA, and the corresponding mRNA primers are "NR4A1_uc001rzt.2_1_1_1 and NR4A1_uc001rzt.2_1_1_2". Click on the primer pair you select; this will take you to a new page with details. To view the primers' properties, click on the number next to the instruction "Click here for primer details," which opens a new page with the primer properties listed.
