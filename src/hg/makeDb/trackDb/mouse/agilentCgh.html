Description

This track displays the probes from the Agilent Catalog Oligonucleotide CGH Microarrays.

Agilent's oligonucleotide aCGH platform is an integrated solution that gives you the resolution and flexibility to perform genome-wide DNA copy number variation profiling without amplification or complexity reduction along with simplified experimental design and ease-of-use protocols to handle genomic samples with full complexity. Probe coverage spans both coding and non-coding regions, with emphasis on well-characterized genes, particularly cancer-relevant genes. Probe design and selection have been carefully optimized and validated in multiple model systems with defined aberrations for maximal sensitivity and specificity.

The track consists of the following subtracks:

Mouse Genome CGH Microarray 244A AMADID 014695

This annotation displays the probe positions for the Mouse Genome CGH 1 x 244K microarray from Agilent Technologies, Inc.
  • 235,000+ coding and non-coding mouse sequences represented
  • 7.8 kb overall median probe spatial resolution (6.2 kb median probe spacing in Refseq genes)
  • 1 x 244K slide format printed using Agilent's 60-mer SurePrint technology

Mouse Genome CGH Microarray 105A AMADID 014699

This annotation displays the probe positions for the Mouse Genome CGH 2 x 105K microarray from Agilent Technologies, Inc. The 105A format has the flexibility to run two samples on a single chip.
  • 98,000+ coding and non-coding mouse sequences represented
  • 19.3 kb overall median probe spatial resolution (15.7 kb median probe spacing in Refseq genes)
  • 2 x 105K slide format printed using Agilent's 60-mer SurePrint technology

Mouse Genome CGH Microarray 44K AMADID 015028

This annotation displays the probe positions for the Mouse Genome CGH 4 x 44K microarray from Agilent Technologies, Inc. The 44K format has the flexibility to run four samples on a single chip.
  • 43,000+ coding and non-coding mouse sequences represented
  • 22.3 kb overall median probe spatial resolution (13.1 kb median probe spacing in Refseq genes)
  • 4 x 44K slide format printed using Agilent's 60-mer SurePrint technology
