Mouse ENCODE Downloads 2007 - 2012

Overview

This page contains links to directories containing raw and processed data for mouse ENCODE data produced as part of the ENCODE production phase (September 2007 - December 2012). Formats are described on the UCSC Browser File Format FAQ. For bulk download, retrieval by FTP is recommended along with rsync. Using rsync, for example "rsync -a -P rsync://hgdownload.gi.ucsc.edu/goldenPath/mm9/encodeDCC/wgEncodeDir/wgEncodeFile ./", has the advantage of starting up where it left off after a failure, when ran again.
