ENCODE Downloads

This page contains links to directories containing raw and processed data for ENCODE data released as part of the ENCODE production phase (September 2007-present). Formats are described on the UCSC Browser File Format FAQ. For bulk download, retrieval by FTP is recommended. All files here are covered by the ENCODE data release policy. Preview of unreleased data for download is available from the Test download server and by FTP. Preview data is made available with the warning that it has not yet been reviewed by UCSC for correctness and is subject to change at any time.

The Analysis Working Group (AWG) is a cross-consortium effort to perform integrated analysis of all the data types based on uniform processing, visit the Integrative Analysis Portal and Quality Metrics pages for more information. Click here to visualize the ENCODE Integrative Analysis Data Hub, then select the Dispay box and Load Selected Hubs button. Metadata for AWG downloads is also available.
