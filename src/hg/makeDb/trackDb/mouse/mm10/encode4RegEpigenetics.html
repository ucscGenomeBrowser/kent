Description

This track displays genome-wide epigenomic signals from 1,178 individual ENCODE experiments, including DNase-seq and ATAC-seq for chromatin accessibility, and ChIP-seq for the histone modifications H3K4me3 and H3K27ac, as well as CTCF binding.

  • DNase-seq and ATAC-seq identify regions of open chromatin commonly associated with regulatory elements such as enhancers, promoters, and insulators.
  • ChIP-seq for the histone modifications H3K4me3 and H3K27ac mark active and poised promoters and enhancers.
  • CTCF ChIP-seq reveals chromatin loop anchors and insulator elements.

Each subtrack represents a single experiment in a specific biosample, displayed as a bigWig signal track. These datasets provide the underlying experimental signals used to generate the corresponding layered summary tracks. Additional datasets measuring epigenomic signals are available at the ENCODE portal.

This composite contains both signal (bigWig) and peak (bigBed) data for each experiment, selectable via the "Data Type" facet in the track settings.

Display Conventions and Configuration

Click a specific biosample type and organ/tissue combination to view available datasets. Subtracks can be further filtered by the epigenetic signal type (ATAC, DNase, CTCF, H3K27ac, and H3K4me3) and life stage of the biosample. Each track is colored based on the type of signal.

Data Access

The ENCODE 4 Regulation data on the UCSC Genome Browser can be explored interactively with the Table Browser or the Data Integrator. For automated download and analysis, the genome annotation is stored in bigWig (signal) and bigBed (peak) files that can be downloaded from our download server. The data may also be explored interactively using our REST API. The original data files are also available from the ENCODE portal.

Signal files may be locally explored using bigWigToWig, e.g.,

bigWigToWig -chrom=chr1 -start=100000 -end=100500 https://encode-public.s3.amazonaws.com/2020/10/16/d60aeb91-36d8-4d24-8d03-4c350a5f5ac3/ENCFF220DSP.bigWig stdout

Peak files may be explored using bigBedToBed, e.g.,

bigBedToBed -chrom=chr1 -start=100000 -end=100500 https://encode-public.s3.amazonaws.com/2020/10/16/a1cbfde2-b570-4c32-8b0f-b1557de8849a/ENCFF209SYF.bigBed stdout

Instructions for downloading these tools can be found here.
